## Overview

Lighthouse Analytics is a cookieless, GDPR-friendly product analytics platform.
Teams instrument events with a single script, then build funnels, retention
charts, and segmented dashboards without exporting raw data to a third party.

## Outcome

- Replaced two paid analytics vendors at the launch customer for ~$18k/year saved.
- Powered a 5x growth in tracked events without horizontal scaling.
- Maintained green Core Web Vitals on every public dashboard route.

## Stack

Next.js App Router with React Server Components for dashboards, a TypeScript
ingestion API on Vercel Edge, PostgreSQL with monthly partitioned event tables,
and Tailwind for the design system.
